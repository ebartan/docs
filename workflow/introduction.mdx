<Card title="n8n" icon="circle-nodes" href="/content/components/card-group">
  n8n is an open-source workflow automation tool that allows users to automate tasks and integrate different services and APIs. It features a visual interface for creating workflows, enabling users to connect various applications, perform data transformations, and automate repetitive tasks without extensive programming knowledge. n8n is highly customizable and can be self-hosted, offering flexibility and control over automation processes.
</Card>

<Card title="Flowise" icon="circle-nodes" href="/content/components/card-group">
  Flowise is an open-source, no-code tool designed to streamline and automate business processes. It allows users to create, customize, and manage workflows visually without requiring programming skills. Flowise supports integration with various applications and services, making it easier to automate tasks and improve operational efficiency. It's ideal for businesses looking to optimize their workflows and reduce manual effort.
</Card>
